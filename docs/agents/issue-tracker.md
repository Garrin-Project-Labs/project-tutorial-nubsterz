# Issue tracker

This project is Discord-first.

- Humans coordinate in the Discord Project Space.
- Friendly work items are called **quests** in chat.
- Durable task metadata lives in `.project/tasks/*.json`.
- GitHub repo/PR history is the backend audit trail when a repo exists: `Garrin-Project-Labs/project-tutorial-nubsterz`.
- GitHub Issues are disabled by default for friend/new-user projects.

When a skill expects an issue tracker, use `.project/tasks` as the local tracker unless a maintainer explicitly asks to publish GitHub Issues later.
